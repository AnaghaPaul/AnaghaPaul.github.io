Unequal Chances

How class and gender shaped survival outcomes during the Titanic disaster

Introduction

The Titanic disaster is often remembered as a tragic accident, but survival was far from random. This project analyzes survival patterns to understand how gender, age, and passenger class influenced the chances of survival.

About the Project

Using the Titanic dataset from Kaggle, this analysis explores how demographic factors affected survival outcomes. What began as a learning exercise evolved into an eye-opening example of how data can reveal social inequalities embedded within historical events.

Dataset

The dataset contains passenger-level information, including survival status, gender, age, and passenger class. These variables enable a detailed comparison of survival rates across different demographic groups.

Analysis & Insights

  • Women had a significantly higher survival rate than men.
  • Younger passengers generally had better survival outcomes.
  • Passengers in higher classes had a clear survival advantage.
  • Third-class men faced the lowest chances of survival.

Key Findings

  • Only 1 in 4 men survived compared to 3 in 4 women.
  • First-class women had the highest survival rates.
  • Social class amplified survival disparities across genders.

Conclusion

This project highlights how survival during the Titanic disaster was shaped by societal norms and class structures rather than pure chance. The findings demonstrate the power of data analysis in uncovering hidden patterns and telling impactful stories about inequality.
