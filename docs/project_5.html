Modern Retail & Order Management Data Warehouse

SQL Server–based data warehouse for a retail company selling bicycles, sports clothing, and components across multiple international markets. Supports sales analysis, customer behavior, product performance, and order lifecycle reporting using a dimensional star schema model.

Project Overview

The data warehouse integrates CRM and ERP source systems, consolidates operational data, and delivers analytics-ready datasets. Role-playing Date dimensions enable time-based analysis across order, shipping, and due dates. Power BI dashboards visualize insights for business decision-making.

Stage 1: Data Warehouse Creation (Medallion Architecture)

Implemented using Bronze, Silver, Gold layers:

Bronze Layer

Raw CRM and ERP data ingested as-is to preserve source fidelity and traceability.

Silver Layer

Data cleansed, standardized, deduplicated, and enriched with reusable reference data (e.g., calendar table).

Gold Layer

Business-ready datasets modeled using star schema with fact and dimension tables, optimized for SQL queries and Power BI reporting.

Benefits: Incremental improvement in data quality, scalable analytics, and faster query performance enabling reliable business insights.
