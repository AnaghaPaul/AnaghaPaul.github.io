Modern Retail & Order Management Data Warehouse

SQL Server–based data warehouse for a retail company selling bicycles, sports clothing, and components across multiple international markets. Supports sales analysis, customer behavior, product performance, and order lifecycle reporting using a dimensional star schema model.

Project Overview

The data warehouse integrates CRM and ERP source systems, consolidates operational data, and delivers analytics-ready datasets. Role-playing date dimensions enable time-based analysis across order, shipping, and due dates. Power BI dashboards visualize insights for business decision-making.

Stage 1: Data Warehouse Creation (Medallion Architecture)

Bronze Layer

Raw CRM and ERP data ingested as-is to preserve source fidelity and traceability.

Silver Layer

Data cleansed, standardized, deduplicated, and enriched with reusable reference data.

Gold Layer

Business-ready datasets modeled using a star schema optimized for analytics and BI.

Benefits: Improved data quality, scalable analytics, and faster query performance enabling reliable business insights.

Stage 2: SQL-Based Data Analysis

Analysis follows five structured stages: Explore → Profile → Clean → Shape → Analyze.

Explore

  • Understand structure, relationships, and anomalies
  • Benefit: Guides transformation planning

Profile

  • Assess statistics and data quality
  • Benefit: Early issue detection

Clean

  • Handle missing values, duplicates, and inconsistencies
  • Benefit: Trustworthy datasets

Shape

  • Transform data into star-schema structures
  • Benefit: Simplified querying and BI integration

Analyze

  • Identify trends, seasonality, and revenue drivers
  • Benefit: Actionable business insights

Stage 3: Power BI Visualization

Star-schema data is visualized in Power BI dashboards with interactive filtering, drill-downs, and KPI tracking across international markets.

Benefit: Clear, decision-ready business intelligence.
