Modern Retail & Order Management Data Warehouse

SQL Server–based data warehouse for a retail company selling bicycles, sports clothing, and components across multiple international markets. Supports sales analysis, customer behavior, product performance, and order lifecycle reporting using a dimensional star schema model.

Project Overview

The data warehouse integrates CRM and ERP source systems, consolidates operational data, and delivers analytics-ready datasets. Role-playing Date dimensions enable time-based analysis across order, shipping, and due dates. Power BI dashboards visualize insights for business decision-making.

Stage 1: Data Warehouse Creation (Medallion Architecture)

Implemented using Bronze, Silver, Gold layers:

Bronze Layer

Raw CRM and ERP data ingested as-is to preserve source fidelity and traceability.

Silver Layer

Data cleansed, standardized, deduplicated, and enriched with reusable reference data (e.g., calendar table).

Gold Layer

Business-ready datasets modeled using star schema with fact and dimension tables, optimized for SQL queries and Power BI reporting.

Benefits: Incremental improvement in data quality, scalable analytics, and faster query performance enabling reliable business insights.

Stage 2: SQL-Based Data Analysis

Data analysis follows five structured stages: Explore → Profile → Clean → Shape → Analyze.

1. Explore

  • Examine datasets to understand structure, relationships, and anomalies.
  • Benefit: Provides an overview of data landscape for planning transformations.

2. Profile

  • Generate descriptive statistics and assess data quality metrics.
  • Benefit: Early detection of inconsistencies ensures reliable analysis.

3. Clean

  • Handle missing/inconsistent data, remove duplicates, standardize formats.
  • Benefit: Produces accurate, trustworthy datasets for downstream analysis.

4. Shape

  • Transform data into analytical-ready structures using fact and dimension tables from the star schema.
  • Benefit: Simplifies querying and integration with BI tools.

5. Analyze

  • Perform exploratory analysis on customer behavior, product performance, and sales trends.
  • Identify seasonality, growth patterns, and revenue drivers.
  • Benefit: Enables actionable business decisions and data-driven strategy.

Stage 3: Power BI Visualization

Star-schema data is visualized in Power BI dashboards for interactive slicing, drill-down, and KPI monitoring. Dashboards highlight products, sales trends, and customer segments across international markets.

Benefit: Clear, interactive view of business performance to support strategic decisions.
